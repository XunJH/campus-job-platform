# 归档说明

这个目录用于存放 `Ai/backend` 下当前不参与主服务启动、但可以临时保留备查的文件。

## 分类

- `logs/`
  - 历史启动日志、重启日志
  - 不参与任何运行逻辑

- `temp-tests/`
  - 一次性测试脚本、手动验证脚本
  - 不参与正式接口或服务启动

## 当前保留在根目录的文件

以下文件仍保留在 `Ai/backend` 根目录，因为它们对日常运行仍有直接用途：

- `app/`
- `.env`
- `.env.example`
- `requirements.txt`

## 启动方式变化

AI 服务原先的两个启动脚本已经移除：

- `Ai/backend/start.bat`
- `Ai/backend/启动服务.bat`

当前统一使用项目根目录的：

- `启动全部服务.bat`
